# 🪄 洛克王国徽章图鉴资源库

本仓库收集整理了《洛克王国》游戏中的各类魔法徽章图片资源，包含**魔法成长徽章(25枚)**和**十大元素徽章(10枚)**。所有图片均按游戏内获取顺序排列，方便玩家收藏和参考。

> 《洛克王国》是由腾讯游戏开发的儿童社区养成类网页游戏，徽章系统是游戏核心成长体系。玩家通过挑战八大徽章守护者获得元素徽章，同时通过等级提升解锁成长徽章，象征着魔法师的成长历程。
